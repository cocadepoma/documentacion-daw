Learn SASS in only 20 minuteSASS turns your CSS into a complete programming language. Learn it in 20 minutes with EDteam. It is a short course, but with enough material to start working with SASS now.

SASS is an acronym for Syntactically Awesome Stylesheets, and refers to a style sheet preprocessor that gives powers to the static CSS language, turning it into a scripting language. Thanks to SASS we have variables, control statements and functions among other features, which are intended to be written without repeating themselves and reusing as much code as possible.

To begin with, we will say that there are two versions of SASS, the old one (with a .sass extension) that used indentation to separate the blocks, and the new one, which uses curly braces just like the standard CSS syntax (with a .scss extension), b and lo which is much more familiar and is what we are going to see in this video.

Finally comes the USB cable that never gets tangled

We still can't live without them, but they are a nightmare when they get tangled up. Magnetic USB cables are the solution to avoid this inconvenience.

Murphy's Law tells us that when we put two perfectly bent and separated cables in a drawer, when we open it again they will be infinitely entangled with each other. And if it's just a cable, when you use it again it will have a knot. Science cannot explain this phenomenon, but it has happened to all of us. luckily, there is a solution.

SuperCalla is a USB cable that never gets tangled. It has caught our attention because of its intelligent design, because it seems that it does what it promises. How does SuperCalla ensure that the USB cable never gets tangled? The key lies in the ring-shaped magnets strategically placed along its entire surface.
